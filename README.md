# Full Stack Web Application of a Travel Website

- Utilize a MEAN Stack development approach to complete a project for a Travel Website
- Use MongoDB, ExpressJS, AngularJS, NodeJS to create the backend and frontend
- Establish RESTful API Endpoints and test with Postman
- Create an additional SPA administrative single page application to allow the manipulation of files after an authenticated login
- Use JWT Authentication for login
- Leverage Git for source code management and continuous integration
